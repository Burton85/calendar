# calendar 
>這是一個使用vue 2 + vuetifyjs 開發的小型專案

## 1.  如何執行這個專案：

##### 方法一：直接開啟此　github page <a href="https://burton85.github.io/calendar/">連結</a>
    
##### 方法二：下載我的git檔案並在本地運行
    
1.下載 git 檔案
    
```
    $ git clone https://github.com/Burton85/calendar.git
```
    
2.進入資料夾
    
```
    $ cd calendar
```
    
3.下載必要依賴
    
```
    $ npm install
 ```
    
4.運行專案
    
```
    $ npm run server
```
    
5.打開 http://localhost:8080 或其他端口
